1. Создать базу данных из скрипта
2. Изменить строку соединения с БД (внизу)
3. Запустить приложение

Реализовано:
- Структура БД
- Добавление 
- Просмотр
- Редактирование
- Удаление

Частично реализована:
- отправка уведомлений из шаблонов

Не реализована:
- Ролевая модель (есть только структура)
- Ролевые ограничя (сейчас просмотр заявок доступен всем)
- Чась функций панели администратора
